### How to setup & run?

* Requirements:
  - NodeJS v20.x

## how to setup?
1. Clone Repository:
```
git clone https://github.com/muhammadabir64/node-tac-toe
```
2. Install Dependencies:
```
npm install
```
3. Start the server:
```
npm start
```
